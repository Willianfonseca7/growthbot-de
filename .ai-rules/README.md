# Engineering Handbook

Este diretório contém todas as regras oficiais de engenharia do GrowthBot DE.

Nenhum agente de IA ou desenvolvedor deve implementar funcionalidades sem considerar estes documentos.

A estrutura está organizada em quatro pilares:

- Foundation
- Engineering
- Product
- Governance

Cada documento possui uma responsabilidade específica e complementa os demais.

O objetivo deste handbook é garantir consistência arquitetural, qualidade técnica, escalabilidade e alinhamento entre produto, engenharia e Inteligência Artificial.
